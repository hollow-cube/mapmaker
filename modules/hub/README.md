# Hub

The core of a mapmaker hub world. Can either be used to create a standalone hub server, or combined with some glue and
the `map` module to create a standalone version of mapmaker.

## Roles

This module manages a hub world and supports map manipulation.

- List maps
- Create a new map
- Edit a map (managed by `map` module)
- Play a map (managed by `map` module)
- Delete a map
